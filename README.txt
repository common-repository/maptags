== Description ==

Maptags is a wordpress plugin to fill checkout address fields dynamically by a maptag. Maptag plugin is fetching address details from https://mapta.gs. To use maptag plugin you must need a mapta.gs account. Goto Maptags in Dashboard, put username and password of https://mapta.gs and submit. Now the plugin is activated.
